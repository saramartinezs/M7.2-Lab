# CSS Layout

## Overview

The purpose of this lab was to experiment with different CSS layout techniques. We were provided with the HTML and some basic CSS. The goal was to achieve the same layout as the provided image. 

### To view the web page
Clone the repository and open in VSCode. Use the Live Preview extension to view the web page. You can either view it within VSCode or in a web browser.
